#在Github上使用Hexo搭建个人博客#

##第一步 环境安装##

  • Nodejs
  • Git（如果有Xcode则不必安装）
  • Hero
    1.安装Nodejs
    https://nodejs.org/en/

2.安装 Git

brew install git

3.安装 Hero

$ sudo npm install -g hexo

如果以上命令安装失败的话，换

sudo npm install –unsafe-perm –verbose -g hexo

##第二步 hexo 配置##

新建blog文件夹，并将它初始化（blog是文件夹的名字，也可以取别的名字）

hexo init blog
cd ~/blog/
npm install

##第三步 git配置##

1.注册github帐号并登录

2.获取本机的ssh口令

1）输入获取代码，回车直到出现图片所示图形为止

$ ssh-keygen

2）输入编译代码

$ vim ~/.ssh/id_rsa.pub
3）出现SSH口令后，将红框部分复制，并在下方输入:q，随后按下回车可以退出该窗口

4）进入到github页面设置SSH口令

step1.点击用户下拉菜单中的settings

step2.点击左侧的SHH and GPG keys

step3.在Title中输入口令名称（随意）

step4.在key中贴上SSH口令

3.与Github账户关联
新建一个名为 [your_account].github.io 的仓库，
username填写github的登录用户名，否则上线的时候会报错,
是否公开选项可以选取Public,
可编辑的README.md文件（建议勾选）,
点击create repository生成仓库完毕;

4.博客设置
打开前面创建的blog目录下的 _config.yml 文件，在修改最下方的deploy为：
*注意，冒号后面一定要加空格

deploy:
type: git
repo: gihub: https://gith https://github.com/[your_account]/[your_accout].github.io.git
branch: master

另外，

其他博客设置

title：博客名称
subtitle：博客副标题
description：博客描述
author：作者
language：语言（简体中文是zh-CN）

4.在终端上传博客（部署到git）
安装hexo的git部署,在命令行中执行：

$ npm install hexo-deployer-git –save

然后，将生成静态页面并部署到github的仓库中，执行：

$ hexo g

最后，上传代码

$ hexo d

当提示 INFO Deploy done: git 即上传成功，这时就可以通过 http:\[your_account].github.io 来访问你的个人站点了。

##新建与更新博客##

  1. 新建博客
    1）终端进入blog文件下，

    $ cd blog
    输入新建博客代码
    $ hexo new ‘filename’

2）此时在bolg/source/_posts/下面会看到新建的博客

3）博客文件的后缀是.md文件，OS下推荐使用MOU编辑器

2.更新博客
完成编辑后，在终端上依次重复以下代码（此时必须先将编辑器关闭，否则会出现获取错误）

$ hexo g
$ hexo d
当提示 INFO Deploy done: git 即上传成功

完成后便能刷新博客网页看到新更新的内容了
